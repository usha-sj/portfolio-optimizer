# portfolio-optimizer
TLDR: A portfolio optimization tool that recommends optimal asset allocations based on historical data and risk preferences.

The definition of portfolio optimisation (according to a quick Google search) is: Portfolio optimization is a quantitative method used in finance to select the best mix of assets for an investment portfolio, aiming to achieve maximum return.
This is a project I started in early 2025 when I started investing for the first time, trying to understand risk and return. More detailed info being updated soon!
